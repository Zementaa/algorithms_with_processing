Dieser Ordner enthält die zum einen im PNG-Ordner die erstellten Bilder zur Reproduktion. Zudem gibt es JSON-Ordner wo die Konfigurationen,
wie im Kpaitel "Anleitung zur Reproduktion der Ergebnisse" beschrieben ist.

Für eine einfache Reproduktion kann der  Pfad, welche sich in den Klammern
 im Filename der PNG's verwendet werden (jeder Schritt mit einem "_" getrannt.
Hierfür kann der Pfad einfach im Start-Fenster des Konfigurator genutzt werden.

1. Menü-"Beispiele"
2. Untermenü-"GRissland"
3. Untermenü- XXX (immer Generator abhängig -->  im Filename beschrieben)
4. Auswahl- XXX (immer die entsprechnde --> im Filename beschrieben)

5. Ausführen.
